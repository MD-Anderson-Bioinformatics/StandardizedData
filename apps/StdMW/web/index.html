Metabolomics Workbench
MDACC Standardized Data Metabolomics Workbench Tool provides a batch effects-centric view of the National Metabolomics Data Repository (NMDR) at the MetabolomicsWorkbench.org data, with options to download data with various settings.
Download Options
Downloaded files are ready to be used by MDA's batch effects tools. Data files are tab delimited, with a leading tab and samples across the top and features (metabolites) down the side.
Data Drop Class - This download will work for most studies. Most studies include an extra 'class' in the data files. This drops that column from the data. You will need to examine the raw data for extra 'class' entries and then use this option.
Data Merge Sample-Class - Some studies include 'class' in the data files, which is necessary to distinguish two samples with the same identifier, but different 'classes'. You will need to examine the raw data for duplicate entries and then use this option.
Data Raw - Some is usable in raw format.
Batch/Factors - Factors for a project downloaded in Batch Effects format. (Format is a header row of 'Samples' followed by the factor names.)
